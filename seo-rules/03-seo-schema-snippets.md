# Advanced SEO: Schema & Snippets

<rules>
1. **JSON-LD Schema Implementation:** Always inject `@context`: `https://schema.org` and validate `@type`. Use `FAQPage`, `Article`, `LocalBusiness`, or `Product` schema where applicable.
2. **The Inverted Pyramid:** When writing informational text blocks, the first 40-50 words MUST directly answer the user's intent to optimize for AI-generated summaries and Featured Snippets.
3. **Data Extraction Formats:** Search engines prefer structured fragments. Use `<ul>`, `<ol>`, and `<table>` tags for lists and comparisons instead of comma-separated text within a paragraph.
</rules>
